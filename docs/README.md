Introducing ZealBlog, a vibrant platform designed to empower users to share their thoughts and insights through the medium of traditional blog posts and dynamic video summaries.
Crafted with a potent blend of HTML, CSS, JavaScript, React.js, Material-UI (MUI), Node.js, Express.js, AWS, and Docker, ZealBlog offers a seamless and immersive experience for both creators and readers.
To get started, ensure Node.js and Docker are installed on your system, then clone the repository, navigate to the project directory, and run npm install to install dependencies. Launch ZealBlog locally
with npm start, enabling you to explore its features, create engaging blog posts, and delve into video summaries. Deployment is made simple with our Docker configuration, while contributions from the community are
warmly welcomed to enhance ZealBlog's functionality and user experience.
